# 秋叶音乐

## 1.0 版本  
功能：

- 在线播放音乐、本地播放音乐
- 音乐搜索功能
- 歌词展示、滑动更新歌曲进度
- 音乐排行版

## 2.0 版本
新增功能：

- 用户登录
- 音乐上传、下载、收藏、缓存
- 音乐歌单（支持自定义）
- 歌词滑动惯性、滑动后可选择是否更新进度
- 设置下一首播放  

修复bug  
　　①:第一次安装播放时会有两首歌曲同时播放  
　　②:排行版之间切换时出现数据错误  
　　③:通知栏在应用退出后依然存在  
　　④:有时候播放信息为null,，但是依然在播放  
　　⑤:修复重新打开音乐播放器时播放服务崩溃
